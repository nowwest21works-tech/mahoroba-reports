# 不動産購入 諸費用シミュレーター

## 構成

```text
tools/real-estate-cost/
├─ pro/index.html
└─ simple/index.html
```

## pro版

**土地購入 決済資金・諸費用シミュレーター Pro**

今西さん実務用。土地決済前に、決済時必要資金・後日発生見込み・自己資金目安を確認するためのHTMLです。

主な機能:

- 売買価格、借入額、評価額割合または固定資産税評価額の直接入力
- 固都税清算金の概算
- 土地所有権移転登記・抵当権設定登記の税率選択
- 仲介手数料の自動計算・なし・手入力
- 住宅ローン返済目安
- 土地固有費用10行
- 不動産取得税の後日見込み
- スマホ上部サマリー
- 印刷/PDF保存

## simple版

**土地代だけで見ていませんか？住まいの総額ざっくり診断**

LINE公式アカウント登録特典用。一般の方向けに、物件種別ごとの総額感と注意点を伝えるHTMLです。

対応タイプ:

- 土地のみ
- 土地＋注文住宅
- 建売住宅
- 中古戸建
- 新築分譲マンション
- 中古マンション

主な機能:

- 購入タイプ別の総額レンジ表示
- 物件種別ごとの注意点表示
- 借入予定額、借入期間、金利、ボーナス払い併用額による住宅ローン返済目安
- 月々返済額、ボーナス加算額、返済額総額の概算表示
- 自己資金チェック
- 個別相談フォームへのCTA

## 注意

本ツールは概算です。最終金額は、売買契約書、固定資産税評価額、司法書士見積、金融機関見積、県税事務所、自治体、管理会社等の確認を優先してください。

## 公的情報の参照先

- 国税庁：登録免許税の税額表
- 国税庁：登録免許税の税率の軽減措置に関するお知らせ
- 愛知県：不動産取得税
- 愛知県：住宅などの不動産取得税の軽減
- 国土交通省：不動産取引に関するお知らせ、仲介手数料
